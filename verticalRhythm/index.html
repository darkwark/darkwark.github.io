Vertical Rhythm Pattern Pack

Vertical Rhythm is a useful thing. Usually I’m using default Photoshop grid to set it up. But this is not the best solution because sometimes I need native grid for other things. In such cases fast created patterns helps me, I don’t usually save them and because of that they are sink into oblivion after switching between different pattern packs and I have to make new pattern again and again (silly, yeah). So, I decided to generate a single pattern pack with popular leading values for daily use in projects.

Download Vertical Rhythm Pack.pat

You have to move it to the Photoshop/Presets/Patterns/ and restart your Photoshop. Then you’ll see it in the list of all pattern sets in the context menu of pattern window

Sizes

In the pattern pack you’ll find following versions of leading: 6, 8, 9, 10, 11, 12, 14, 18, 20, 24, 25, 30, 35, 36, 45, 48, 50, 55, 60, 72. Simply, these are values of leading from native font panel in Photoshop and some additional values that I use sometimes

Using

The best way to use these patterns will be createing Pattern Fill Layer (Layer → New Fill Layer → Pattern…), choosing apropriate pattern for your leading and setting up opacity for the layer
Patterns have names in the format 20px leading, therefore it would be better if you’ll switch view of the patterns window to Large List, you can do it in the context menu of the pattern panel
At the result you’ll have black lines with selected height. If you want to use it on the dark background, just switch blending mode to Divide this will change color of lines to white

Yup

That’s it for today. Hope these patterns will help you in your work. I made them via script by the way, I just have an array of all sizes and script generates all patterns. So if you have suggestions for other sizes — write them down in comments and I’ll update the file
